# Changelog

All notable changes to Mechanisms are recorded here.

## Unreleased

- Added `/mech unlockrecipes [player]` to open registered Mechanisms recipes in the recipe book manually.

## 0.1.0 - 2026-06-19

### Added

- Initial local Paper/Purpur 26.2 plugin baseline.
- Logistics module with extractor, pipe, fast pipe, express pipe, router, inserter, overflow, and trash blocks.
- Transactional item transfer, route search, filters, channels, upgrades, and pending-transfer recovery.
- `/mech` command surface for menus, recipes, diagnostics, stats, debug tools, and selftest.
- Local Java 25 Gradle build and `server-dev` validation workflow.
